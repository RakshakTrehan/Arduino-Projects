# LED Blink

## Overview

This project demonstrates the basics of controlling an LED using an Arduino Uno. The LED blinks continuously at a fixed interval.

## Components Used

- Arduino Uno
- LED
- 220Ω Resistor

## Concepts Learned

- Digital Output
- pinMode()
- digitalWrite()
- delay()

## Files

- `LED_Blink.ino`
- `circuit.png`

## Future Improvements

- Control blink speed using a potentiometer.
- Replace `delay()` with `millis()` for non-blocking execution.
