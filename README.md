# Arduino Projects

A collection of Arduino-based embedded systems projects demonstrating digital electronics, analog sensing, PWM control, sensor interfacing, and automation using **Arduino Uno** and **Tinkercad**.

---

## 📂 Projects

| No. | Project | Description |
|:---:|---------|-------------|
| 01 | [LED Blink](01_LED_Blink) | Basic LED control using digital output. |
| 02 | [RGB Traffic Light](02_RGB_Traffic_Light) | RGB LED sequencing with timing control. |
| 03 | [Push Button Controlled LED](03_Push_Button_Controlled_LED) | LED control using digital input and push button logic. |
| 04 | [PWM Brightness Control](04_PWM_Brightness_Control) | LED brightness control using PWM and a potentiometer. |
| 05 | [Alert System](05_Alert_System) | LED and buzzer based visual and audible alert system. |
| 06 | [Smart Desktop Companion Lamp](06_Smart_Desktop_Companion_Lamp) | Automatic LED control using ambient light sensing with an LDR. |
