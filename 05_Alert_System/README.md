# Alert System using Arduino Uno

## Overview
This project demonstrates a simple alert system using an Arduino Uno, an LED, and a piezo buzzer. The LED provides a visual indication while the buzzer generates an audible alert. Both components blink together at fixed intervals.

## Components Used
- Arduino Uno
- LED
- 220Ω Resistor
- Piezo Buzzer
- Jumper Wires

## Pin Connections
| Component | Arduino Pin |
|-----------|-------------|
| LED | D13 |
| Buzzer | D9 |

## Concepts Learned
- Digital Output
- pinMode()
- digitalWrite()
- delay()
